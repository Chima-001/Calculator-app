# Modern Calculator App

A fully functional, responsive calculator web app with:

## Features
- Basic arithmetic (+, -, ×, ÷)
- Percentage calculations
- Clear (AC), Delete (⌫)
- Error handling
- Dark modern theme
- Fully responsive (mobile/desktop)
- Accessible (ARIA labels)
- Secure evaluation (no direct eval)

## How to Run
1. Open `calculator.html` in any modern browser
2. Or serve via live server (`npx live-server` or VSCode Live Server extension)

## Tech
- Vanilla HTML/CSS/JS
- CSS Grid + Flexbox
- Secure Function-based expression eval
